Обо мне

Привет! Я веб-разработчик с опытом работы более 3 лет. Моя страсть — создавать современные, удобные и адаптивные веб-сайты. Я специализируюсь на фронтенд-разработке с использованием новейших технологий.

Мой опыт

За время своей работы я успешно завершил множество проектов для различных клиентов, от стартапов до крупных компаний. Я отлично разбираюсь в HTML, CSS, JavaScript

Мое образование

Я прошел профессиональный курс по веб-разработке и дизайну от университета Иннополис. Это помогло мне развить навыки, необходимые для успешной работы в области разработки.

Навыки

HTML

HTML — язык гипертекстовой разметки, который используется для создания структуры веб-страниц.

CSS

CSS — каскадные таблицы стилей, используются для оформления и позиционирования элементов на странице.

JavaScript

JavaScript — язык программирования для создания динамических и интерактивных элементов на веб-страницах.

PHP

PHP - серверный язык программирования, используемый для создания динамических веб-страниц и взаимодействия с базами данных.

Git

Git - система управления версиями, позволяющая отслеживать изменения в коде, сотрудничать с другими разработчиками и управлять различными версиями проектов.
